# TLD Catalog

## Thematic TLD Pairings by Project Type

Use when generating names in brainstorm mode (Step 7) or Track B alternatives (Step 4b). Match the project type to the context gathered in Steps 2 and 7a. Note: `.gg` routes to Dynadot (not name.com). See `references/registrar-routing.md` for all routing rules.

**Removed TLDs:** `.bot`, `.play`, `.buy`, `.sound` — these are not open for public registration (brand TLDs or inactive).

| Project Type | Recommended TLDs |
|---|---|
| Dev tools | `.dev`, `.tools`, `.codes`, `.build`, `.run`, `.sh` |
| Creative | `.studio`, `.design`, `.art`, `.gallery`, `.ink`, `.media` |
| AI products | `.ai`, `.chat` |
| Food / drink | `.coffee`, `.cafe`, `.menu`, `.pizza`, `.recipes`, `.kitchen` |
| Gaming | `.gg`, `.game`, `.quest`, `.lol` |
| Business | `.co`, `.ventures`, `.supply`, `.agency`, `.inc`, `.capital` |
| Community | `.community`, `.social`, `.club`, `.group`, `.team` |
| Commerce | `.shop`, `.store`, `.market`, `.deals` |
| Education | `.academy`, `.school`, `.courses`, `.training` |
| Health | `.health`, `.care`, `.clinic`, `.fit` |
| Music | `.music`, `.band`, `.audio`, `.fm` |
| Finance | `.money`, `.finance`, `.fund`, `.tax`, `.investments` |

## Domain Hack Catalog

Domain hacks break a word across the name and TLD so the whole thing reads as one word. Use as inspiration — verify the ccTLD accepts open registrations before suggesting.

**ccTLD restrictions and registrar routing:** Some ccTLDs are not on name.com and route to fallback registrars. See `references/registrar-routing.md` for the full routing table. Key notes:

- `.er` — **Non-registrable.** No major registrar supports open registration. Still suggest hacks for inspiration but flag the limitation.
- `.al` — **Non-registrable.** Specialty/regional registrars only.
- `.st`, `.ly`, `.is`, `.to`, `.pt`, `.my` — **Route to Dynadot** (not on name.com).
- `.gg` — **Routes to Dynadot** (not on name.com).
- `.nu` — **Routes to Dynadot** (not on name.com).
- All others below are on name.com.
